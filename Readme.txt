1. Requirements

Make sure you have Python 3.9+ installed.

Install the required libraries:

    pip install -r requirements.txt

If you do not have a requirements file, install manually:

    pip install streamlit mazelib numpy matplotlib seaborn

(Include any additional libraries your code uses.)

To install Mazelib do:

import sys
!{sys.executable} -m pip install git+https://github.com/john-science/mazelib.git

2. Running the Application

Navigate to the project folder in your terminal and run:

    streamlit run app.py

Replace "app.py" with the name of your Streamlit file if different.

The Streamlit interface will open automatically in your browser.
